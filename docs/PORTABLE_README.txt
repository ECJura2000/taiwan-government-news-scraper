各機關新聞整理
================

1. 請將整個「各機關新聞」資料夾解壓縮到桌面或其他可寫入的位置。
2. 一般使用者可直接開啟「各機關新聞整理」執行檔。
3. Codex、排程或命令列可使用：

   Windows：各機關新聞整理.exe --headless --json-summary
   Linux/macOS：./各機關新聞整理 --headless --json-summary

4. Excel 會放在「新聞搜集區」，JSON 報告會放在其下的「執行紀錄」。
5. 國土管理署來源需要系統已安裝 Chrome 或 Chromium。
6. 執行檔尚未使用 Windows Authenticode 或 Apple Developer ID 簽章，首次開啟時作業系統可能顯示安全警告。

不要將 webhook、密碼或其他憑證寫入 settings.json。
